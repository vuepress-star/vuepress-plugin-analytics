### As Easy as 1, 2, 3

:::: code-group

::: code-group-item NPM

```bash
# install in your project
npm install -D @starzkg/vuepress-plugin-analytics

# create a markdown file
echo '# Hello VuePress Plugin Analytics' > README.md

# start writing
npx vuepress dev

# build to static files
npx vuepress build
```
:::
